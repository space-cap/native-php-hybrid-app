# 기능 로드맵

이 문서는 Native PHP Hybrid App v1.0에서 구현할 기능 범위를 정리합니다.

## 1단계 - 인증

- 로그인
- 로그아웃
- 회원가입
- JWT 인증

## 2단계 - 게시판

- 게시글 목록
- 게시글 상세
- 게시글 등록
- 게시글 수정
- 게시글 삭제

## 3단계 - 파일

- 이미지 업로드
- 프로필 사진

## 4단계 - Android

- WebView
- 뒤로가기 처리
- 파일 업로드
- Pull To Refresh
- 네트워크 오류 처리

## 5단계 - 서버

- Apache Rewrite
- Composer
- PDO
- SQL Injection 방지
- XSS 방지
- CORS 처리
- 환경설정 `.env`

## v1.0 완료 기준

v1.0은 사용자가 Android 앱에서 회원가입 및 로그인 후 게시판 기능과 이미지 업로드 기능을 사용할 수 있고, 백엔드 API는 Postman으로 검증 가능한 상태를 완료 기준으로 합니다.
