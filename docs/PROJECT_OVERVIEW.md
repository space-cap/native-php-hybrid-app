# 프로젝트 개요

## Native PHP Hybrid App

Native PHP Hybrid App은 Android WebView 기반 하이브리드 앱과 Native PHP REST API를 함께 사용하는 샘플 프로젝트입니다.

이 프로젝트의 목표는 단순히 로그인 화면만 구현하는 예제가 아니라, 실무 프로젝트의 기본 구조를 갖춘 템플릿을 만드는 것입니다. 인증, 게시판, 파일 업로드, Android WebView 연동, 서버 설정까지 포함하여 면접이나 포트폴리오에서 전체 흐름을 시연할 수 있는 수준을 지향합니다.

## 주요 특징

- Native PHP 8 기반 백엔드
- Android WebView 기반 하이브리드 앱
- REST API 설계
- JWT Authentication
- Apache Rewrite
- Composer 기반 패키지 관리
- MariaDB 데이터베이스
- 파일 업로드

## 프로젝트 목표

1. Native PHP만으로 REST API 구조를 명확하게 구성합니다.
2. Android WebView 앱에서 PHP 백엔드 API를 사용하는 흐름을 보여줍니다.
3. 인증, 게시판, 파일 업로드처럼 실무에서 자주 사용하는 기능을 포함합니다.
4. Docker, Git, GitHub, Postman을 활용한 개발 및 검증 흐름을 문서화합니다.
5. 이후 Laravel, Spring Boot 등 다른 프레임워크로 확장하더라도 적응하기 쉬운 계층형 구조를 유지합니다.
